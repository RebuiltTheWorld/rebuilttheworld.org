# RTW Internet Architecture
## Three-Layer Domain System
### Version: v1.0
### Status: Stable
### Scope: Standalone architecture document

---

## Purpose

This document defines the complete internet-facing architecture of the **Rebuilt The World** ecosystem as a **three-layer system**.

It exists as a standalone reference file.

Its purpose is to make clear:

- which domain represents which layer of the ecosystem,
- what is public and what is protected,
- how access and visibility should be controlled,
- how the website, platform, LAB, archive, and technical infrastructure are separated,
- how the system can grow without collapsing into one mixed surface.

This file is intentionally independent from other existing documents.

It does **not** modify previous files.
It introduces a separate, self-contained architectural map.

---
