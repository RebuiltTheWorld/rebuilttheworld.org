# Web Assets License Notes
## Rebuilt The World
### Version: v0.1

---

## Purpose

This document records the intended usage status of visual web assets stored in the `web/` directory, including logo files and related design materials.

It exists to separate:

- code and documentation licensing,
- content licensing,
- and identity asset usage rules.

---

## Scope

This file applies to assets stored inside:

- `web/assets/`
- `web/assets/logo/`

including logo files, exported marks, and related visual identity materials unless otherwise stated.

---

## Logo status

The Rebuilt The World logo and related identity marks are brand assets.

Unless explicitly stated otherwise, they are **not** released into the public domain and are **not** automatically covered by open content permissions.

They may be viewed, referenced, and displayed in connection with the official Rebuilt The World project, repository, website, and related documentation.

Reuse, redistribution, modification, commercial use, or derivative brand use should require explicit permission from the project owner.

---

## Website planning documents

Files such as:

- `web/blueprint.md`
- `web/homepage-content.md`
- `web/assets/logo/rebuilt-the-world-logo-notes.md`

are planning and documentation materials.

Their licensing status should follow the repository’s documentation/content rules unless a more specific rule is later added.

---

## Trademark and identity note

The name **Rebuilt The World**, its logo, and associated visual identity elements should be treated as protected brand identifiers.

This file does not replace the repository-level trademark file.
It only clarifies local intent for assets stored in the `web/` planning layer.
